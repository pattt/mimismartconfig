Описание универсальных скриптов
	Автоматизация
		- Автоматическая регулировка яркости диммеров по уровню движения, auto_ilum.txt  
		- Автоматическое включение или отключение через заданное время одного или несколько элементов, auto_on_or_off.txt 
		- Управление шторами по освещенности, и/или освещению, и/или времени, blinds_bylight.txt 
		- Управление вытяжкой по влажности, byhumidity.txt
		- Управление вытяжкой по свету или геркону, bylight.txt  		
		- Включение элемента на заданное время по нажатии на кнопку, ring.txt
	Климат
		- Проверка возможности включения кондиционера в зависимости от даты, klimatCalendar.txt  
		- Проверка возможности включения кондиционера при низкой температуре на улице, klimatCold.txt  
		- Климат-контроль, кондиционер+отопление, KlimatControl.txt
	Дата/время
		- Индикация отопительного сезона, heating_season_indication.txt  
		- Выполнение действий по времени (с условиями), time.txt
		- Закрытие штор по времени суток и дню недели, blinds_closing_bytime.txt
	Диспетчеризация
		- Скрипт счетчика меркурий, mercury.txt
		- Скрипт сетевого анализатора (расширенные параметры), net_analyzer_pro.txt
		- Скрипт сетевого анализатора (основные параметры), net_analyzer_simple.txt
		- Счетчик импульсов, tic_counter.txt
	Инженерные системы
		- Включение/Выключение элементов по гистерезису, gister.txt
		- Вывод числовых значений от датчика на интерфейс (для тестирования), num2interface.txt
	Групповые действия
		- Действия по изменению состояния устройств, change_state.txt  		
		- Груповое управление шторами, groups_control_blinds.txt
		- Включить все, выключить все (Я ушел, я пришел)(+сообщения на мультирум, проверка герконов, сигналы на ИК), on-off-simple.txt 
		- Выключить все, включить то, что было включено, off_all_on_some.txt 
		- Hабор действий с паузами между ними (+ИК команды), set_action_with_pause.txt
	GSM
		- Проверка баланса модуля GSM, balance.txt
		- Отправка ММS, mms.txt
	Освещение
		- Мигание,blinks.txt  		
		- Включение освещения при длительном движении, on_by_continuous_motion.txt
		- Плавное перетекание RGB подсветки, rgb.txt               
		- Управление питанием подсветки, rgb_pw.txt 
		- Умный перелив лент RGB, rgb-fusion.txt          
		- Умный перелив лент RGB(перелив "вручную", не самой RGB), rgb-fusion-mod.txt 
	Мультирум
		- Будильник, alarm-clock.txt
		- Включение проигрывания с оптического входа медиаточки, optic_in.txt
		- Включение проигрывания с микрофона, microphone.txt
		- Приветствие (Досвидание), salutation.txt
		- Движение музыки по датчикам движения, music_moving(sync).txt
		- Кнопки (с подсветкой) play/pause,volume+(удержание) и next,volume-(удержание) для медиаточки, play_pause_next_vol.txt
	Уведомления
		- Уведомление по SMS, мультирум, звонок, интерфейс (+повторные уведомления), notifications.txt
	Безопасность
		- Информирование о протечке воды, alarm-leak.txt  
		- Информирование о протечке воды (упрощенный вариант), alarm-leak_simple.txt
		- Перекрытие клапана газа, alarm-methane.txt
		- Главный скрипт системы охраны, guard-main.txt  
		- Описание датчиков одной зоны для скрипта охраны, guard-zone.txt  
		- Имитация присутствия, imitation.txt  
		- Блокировка состояния элемента,item-protect.txt  
		- Контроль выхода за диапазон значения датчиков, sensor_out_range.txt  
		- Тестовое срабатывание ежедневное (еженедельное), test_action_everyday_or_week.txt  
		- Тестовое срабатывание ежемесячное, test_action_everymonth.txt
	Прочее
		- Выполнение действия с задержкой, action_with_delay.txt 
		- Управление несколькими исполнителями в зависимости от кол-ва нажатий на выключатель, button.txt
		- Циклическое включение 1го из от 2х до 5ти элементов, cyclically_on.txt		
		- Cкрипт который в зависимости от времени выполняет разные действия, diff_act_bytime.txt  
		- Переключение между пресетами камеры, go_ms.txt  
		- Детекция бездействия исполнителей, idle.txt  
		- Действия по нажатию кнопки домофона (звонок и мультирум), intercom_actions.txt  
		- Действия по включению, on_init.txt
		- Самодиагностика, selftesting.txt  
		- Набор условий и действия по ним (например: если ((a<10) и (b!=4) или  (с==0)) то {действие1;действие2} иначе {действие3;действие4}), set_conditions.txt
